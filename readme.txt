== Description ==

You will be able to Visible Stock quantity under each product in the shop, category and archive pages (wherever there is a loop).

== Installation ==

1. Upload the folder `woo-stock-visible` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To display the Woo Stock Visible Settings, go to WooCommerce Settings > products > inventory.
4. That's it!
